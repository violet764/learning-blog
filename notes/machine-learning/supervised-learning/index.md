# 监督学习

监督学习是机器学习中最基础、应用最广泛的学习范式。给定带标签的训练数据，学习输入到输出的映射关系，从而对未知数据进行预测。

## 📚 模块结构

监督学习模块包含以下五个子模块：

### [线性模型](./linear-models/index.md)

线性模型假设目标与特征之间存在线性关系，是监督学习的基石。

- **线性回归**：回归任务的基础，OLS 解析解与梯度下降
- **逻辑回归**：二分类经典算法，Sigmoid 与交叉熵
- **正则化**：L1/L2 正则化，防止过拟合

### [支持向量机](./svm/index.md)

SVM 寻找最大间隔超平面，通过核技巧处理非线性问题。

- **SVM 理论**：最大间隔、对偶问题、KKT 条件
- **核方法**：核技巧、Mercer 定理、常用核函数
- **实现细节**：SMO 算法、大规模数据处理

### [树模型与集成学习](./tree-models/index.md)

基于决策树的模型家族，集成学习方法显著提升性能。

- **决策树**：分裂准则、剪枝策略
- **随机森林**：Bagging 思想、OOB 误差
- **Boosting 家族**：AdaBoost、GBDT、XGBoost、LightGBM

### [贝叶斯方法](./bayesian-methods/index.md)

基于贝叶斯推断的方法，自然量化预测不确定性。

- **朴素贝叶斯**：条件独立性假设、三种变体
- **EM 算法**：隐变量模型、高斯混合模型
- **高斯过程**：贝叶斯非参数方法、贝叶斯优化

### [理论框架](./framework/supervised-learning.md)

监督学习的数学理论基础，涵盖学习理论的核心概念。

- **统计学习理论**：ERM 原则、泛化误差
- **模型选择**：偏差-方差权衡、正则化理论
- **评估方法**：交叉验证、统计检验

## 🔗 模块对比

| 模块 | 核心思想 | 优势 | 典型应用 |
|------|----------|------|----------|
| 线性模型 | 线性组合 | 简单、可解释 | 回归预测、二分类 |
| SVM | 最大间隔 | 泛化能力强 | 文本分类、图像识别 |
| 树模型 | 递归分裂 | 非线性、可解释 | 表格数据竞赛 |
| 贝叶斯 | 概率推断 | 不确定性量化 | 小样本、在线学习 |

## 📖 学习路径

```
理论框架 ──→ 线性模型 ──→ SVM ──→ 树模型 ──→ 贝叶斯方法
     │            │          │          │            │
     ▼            ▼          ▼          ▼            ▼
  ERM原理      线性回归    最大间隔    决策树      朴素贝叶斯
  泛化误差     逻辑回归    核技巧      随机森林    EM 算法
  偏差方差     正则化      SMO 算法    XGBoost     高斯过程
```

### 推荐学习顺序

**初学者路径**：
1. 理论框架 → 2. 线性模型 → 3. 树模型（决策树）→ 4. 贝叶斯方法（朴素贝叶斯）

**进阶路径**：
1. SVM → 2. 树模型（集成学习）→ 3. 贝叶斯方法（高斯过程）

**竞赛/实践路径**：
1. 线性模型（基线）→ 2. 树模型（XGBoost/LightGBM）→ 3. 模型融合

## 🎯 核心概念

### 监督学习问题定义

给定训练集 $D = \{(\mathbf{x}_i, y_i)\}_{i=1}^n$，学习映射 $f: \mathcal{X} \to \mathcal{Y}$：

- **回归**：$\mathcal{Y} \subseteq \mathbb{R}$（连续值）
- **分类**：$\mathcal{Y} = \{1, 2, \ldots, K\}$（离散标签）

### 经验风险最小化

$$ \hat{f} = \arg\min_{f \in \mathcal{F}} \frac{1}{n} \sum_{i=1}^n L(f(\mathbf{x}_i), y_i) $$

### 偏差-方差分解

$$ \mathbb{E}[(\hat{y} - y)^2] = \text{Bias}^2 + \text{Variance} + \sigma^2 $$

## 💡 算法选择指南

| 数据特点 | 推荐算法 | 原因 |
|----------|----------|------|
| 线性关系 | 线性回归/逻辑回归 | 简单高效、可解释 |
| 小样本高维 | SVM | 最大间隔保证泛化 |
| 表格数据 | XGBoost/LightGBM | 性能最优 |
| 需要不确定性 | 贝叶斯方法 | 概率输出 |
| 可解释性优先 | 决策树/线性模型 | 决策路径清晰 |
