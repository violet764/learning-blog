# 贝叶斯方法

贝叶斯方法是机器学习中一类重要的建模方法，其核心思想是将模型参数视为随机变量，通过概率推断来更新对参数的认知。与频率学派方法不同，贝叶斯方法能够自然地量化预测的不确定性，并在小样本场景下表现出色。

## 📚 本模块内容

本模块介绍三种核心的贝叶斯方法：

### [朴素贝叶斯分类器](./naive-bayes.md)

朴素贝叶斯是最简单实用的贝叶斯分类方法，基于特征条件独立性假设。

**核心内容**：
- 贝叶斯定理与条件独立性假设
- 高斯朴素贝叶斯、多项式朴素贝叶斯、伯努利朴素贝叶斯
- 拉普拉斯平滑处理零概率问题
- 完整的Python实现与应用示例

**适用场景**：文本分类、垃圾邮件过滤、情感分析

### [EM算法](./em-algorithm.md)

期望最大化算法是处理隐变量模型的经典方法，通过迭代优化边缘似然。

**核心内容**：
- 隐变量与边缘似然的计算困难
- Jensen不等式与ELBO下界函数
- E步与M步的数学推导
- 高斯混合模型（GMM）的EM求解
- 从零实现EM算法

**适用场景**：高斯混合模型、隐马尔可夫模型、缺失数据填补

### [高斯过程](./gaussian-processes.md)

高斯过程是贝叶斯非参数方法的代表，直接在函数空间进行推断。

**核心内容**：
- 高斯过程的定义与核心思想
- 预测分布的推导
- 核函数的选择与组合
- 高斯过程回归与分类
- 贝叶斯优化应用

**适用场景**：小样本回归、不确定性估计、贝叶斯优化

## 🔗 方法对比

| 特性 | 朴素贝叶斯 | EM算法 | 高斯过程 |
|------|------------|--------|----------|
| 类型 | 参数化 | 参数化 | 非参数化 |
| 参数估计 | 解析解 | 迭代优化 | 解析解/核技巧 |
| 不确定性 | 类别概率 | 隐变量分布 | 完整后验分布 |
| 训练速度 | 快 | 中等 | 慢（大数据） |
| 适用数据量 | 任意 | 中等 | 小到中等 |
| 主要优势 | 简单高效 | 处理隐变量 | 不确定性量化 |

## 📖 学习路径

```
朴素贝叶斯 ──→ EM算法 ──→ 高斯过程
    │              │            │
    ▼              ▼            ▼
 离散特征      隐变量模型    连续函数
 文本分类      混合模型      贝叶斯优化
```

建议学习顺序：
1. 从**朴素贝叶斯**入手，理解贝叶斯推断的基本思想
2. 学习**EM算法**，掌握处理隐变量的迭代方法
3. 深入**高斯过程**，体验贝叶斯非参数方法的强大

## 🎯 核心概念

### 贝叶斯推断的核心公式

$$ P(\theta|D) = \frac{P(D|\theta)P(\theta)}{P(D)} \propto P(D|\theta)P(\theta) $$

- **先验** $P(\theta)$：参数的初始信念
- **似然** $P(D|\theta)$：数据如何由参数生成
- **后验** $P(\theta|D)$：观测数据后更新的信念

### 频率学派 vs 贝叶斯学派

| 视角 | 参数 | 点估计 | 不确定性 |
|------|------|--------|----------|
| 频率学派 | 未知常数 | MLE/MAP | 需要额外方法 |
| 贝叶斯学派 | 随机变量 | 后验均值/众数 | 自然包含 |
