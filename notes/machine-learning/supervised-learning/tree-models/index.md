# 树模型与集成学习

决策树是一类基于树结构进行决策的机器学习算法，具有直观、可解释性强的特点。集成学习通过组合多个弱学习器来构建强学习器，显著提升了模型的预测性能，是现代机器学习中最重要的技术之一。

## 📚 本模块内容

### [决策树](./decision-trees.md)

树模型的基础，通过递归分裂构建预测模型。

**核心内容**：
- 分裂准则：信息增益、信息增益率、基尼系数
- ID3、C4.5、CART 算法对比
- 剪枝策略：预剪枝与后剪枝
- 回归树的分裂准则

### [随机森林](./random-forest.md)

基于 Bagging 思想的集成方法，通过自助采样和特征随机化提升泛化能力。

**核心内容**：
- Bagging 原理与自助采样
- 特征随机化与多样性
- 袋外误差（OOB）估计
- 特征重要性评估

### [AdaBoost](./adaboost.md)

基于 Boosting 思想的经典算法，通过调整样本权重逐步改进模型。

**核心内容**：
- Boosting 的核心思想
- 样本权重更新机制
- 弱分类器权重计算
- 理论保证：训练误差上界

### [梯度提升树](./gradient-boosting.md)

将提升问题转化为梯度下降优化，是 Boosting 方法的通用框架。

**核心内容**：
- 函数空间梯度下降
- 损失函数的选择
- 学习率与迭代次数的平衡
- 正则化策略

### [XGBoost](./xgboost.md)

梯度提升的高效工程实现，引入二阶优化和正则化改进。

**核心内容**：
- 二阶泰勒展开近似
- 正则化目标函数
- 列采样与行采样
- 并行化与缓存优化

### [LightGBM](./lightgbm.md)

微软提出的高效梯度提升框架，针对大规模数据的优化方案。

**核心内容**：
- 直方图算法加速
- GOSS（梯度单边采样）
- EFB（互斥特征捆绑）
- Leaf-wise 生长策略

## 🔗 方法对比

### 单模型 vs 集成模型

| 特性 | 决策树 | 随机森林 | GBDT |
|------|--------|----------|------|
| **集成方式** | 无 | Bagging | Boosting |
| **可解释性** | 高 | 中 | 低 |
| **过拟合风险** | 高 | 低 | 中 |
| **并行化** | - | 容易 | 困难 |
| **调参难度** | 低 | 低 | 中 |

### GBDT 框架对比

| 特性 | 传统 GBDT | XGBoost | LightGBM |
|------|-----------|---------|----------|
| **优化方法** | 一阶梯度 | 二阶泰勒 | 二阶泰勒 |
| **正则化** | 无 | L1+L2 | L1+L2 |
| **缺失值处理** | 需预处理 | 自动处理 | 自动处理 |
| **生长策略** | Level-wise | Level-wise | Leaf-wise |
| **大规模数据** | 慢 | 中 | 快 |

## 📖 学习路径

```
决策树 ──→ 随机森林 ──→ AdaBoost ──→ GBDT ──→ XGBoost ──→ LightGBM
   │          │            │           │          │            │
   ▼          ▼            ▼           ▼          ▼            ▼
分裂准则    Bagging     Boosting    梯度下降   二阶优化    直方图算法
剪枝策略    OOB估计     权重更新    损失函数   正则化      GOSS/EFB
```

建议学习顺序：
1. 从 **决策树** 入手，理解树模型的分裂和剪枝
2. 学习 **随机森林**，掌握 Bagging 集成思想
3. 通过 **AdaBoost** 理解 Boosting 的基本原理
4. 深入 **GBDT**，理解梯度提升框架
5. 学习 **XGBoost** 和 **LightGBM** 的工程优化

## 🎯 核心概念

### Bagging vs Boosting

| 方面 | Bagging | Boosting |
|------|---------|----------|
| **基学习器** | 并行独立训练 | 串行依赖训练 |
| **样本权重** | 相等 | 逐步调整 |
| **组合方式** | 投票/平均 | 加权求和 |
| **降低** | 方差 | 偏差 |
| **代表算法** | 随机森林 | GBDT、AdaBoost |

### 偏差-方差权衡

- **Bagging**：通过平均多个模型降低方差，适合高方差低偏差的基学习器
- **Boosting**：通过逐步修正残差降低偏差，适合高偏差低方差的基学习器

## 💡 适用场景

**适合使用树模型**：
- 特征为混合类型（连续、离散）
- 需要可解释性
- 数据存在非线性关系

**适合使用集成方法**：
- 追求高预测精度
- 数据量充足
- 计算资源允许

**选择建议**：
- 小规模数据：决策树、随机森林
- 中等规模：XGBoost
- 大规模数据：LightGBM
- 需要可解释性：决策树、SHAP 分析
