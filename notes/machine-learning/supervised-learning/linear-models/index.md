# 线性模型

线性模型是监督学习的基石，假设目标变量与特征之间存在线性关系。虽然形式简单，但具有**可解释性强**、**计算高效**、**理论基础完善**等优点，是理解更复杂模型的起点。

📌 **线性模型家族**：线性回归（回归任务）、逻辑回归（分类任务）、正则化方法（防止过拟合）

## 概述

线性模型的核心假设是目标与特征之间存在线性组合关系：

$$y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \cdots + \beta_p x_p$$

根据任务类型和目标函数的不同，衍生出多种线性模型：

| 模型 | 任务 | 输出范围 | 损失函数 |
|------|------|----------|----------|
| 线性回归 | 回归 | $(-\infty, +\infty)$ | 均方误差 |
| 逻辑回归 | 二分类 | $(0, 1)$ 概率 | 交叉熵 |
| Softmax 回归 | 多分类 | 概率分布 | 交叉熵 |

## 快速对比

### 线性回归 vs 逻辑回归

| 方面 | 线性回归 | 逻辑回归 |
|------|----------|----------|
| **任务类型** | 回归 | 分类 |
| **输出** | 连续值 | 概率 (0-1) |
| **激活函数** | 恒等函数 | Sigmoid |
| **损失函数** | MSE | 交叉熵 |
| **解析解** | ✅ 存在 | ❌ 不存在 |
| **假设** | 线性关系 | 线性对数几率 |

### 正则化方法对比

| 特性 | Ridge (L2) | Lasso (L1) | Elastic Net |
|------|------------|------------|-------------|
| **特征选择** | ❌ | ✅ | ✅ |
| **处理共线性** | ✅ | ⚠️ | ✅ |
| **计算效率** | 高 | 中 | 中 |
| **稳定性** | 高 | 较低 | 高 |

## 学习路径

```
线性回归 ──→ 理解最小二乘法 ──→ 掌握梯度下降
    │
    ▼
逻辑回归 ──→ 理解 Sigmoid ──→ 掌握交叉熵
    │
    ▼
正则化 ──→ 理解过拟合 ──→ 掌握 L1/L2 ──→ 实践 Elastic Net
```

## 适用场景

**适合使用线性模型**：
- 特征与目标存在近似线性关系
- 需要模型可解释性
- 数据量较小或中等
- 作为基线模型

**不适合使用线性模型**：
- 特征与目标存在复杂非线性关系
- 需要捕捉特征交互
- 数据存在复杂的层次结构

💡 **提示**：即使最终使用复杂模型，线性模型也是重要的基线参照。
