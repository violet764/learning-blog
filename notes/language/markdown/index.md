## 📝 什么是 Markdown？

Markdown 是一种轻量级标记语言，排版语法简洁，让人们更多地关注内容本身而非排版。它使用易读易写的纯文本格式编写文档，可与 HTML 混编，可导出 HTML、PDF 以及本身的 `.md` 格式的文件。

因**简洁**、**高效**、**易读**、**易写**而被大量使用，成为编写文档、博客、技术说明的首选格式。

## 💡 使用技巧

1. **保持简洁**：避免过度复杂的嵌套结构
2. **合理使用空行**：提高可读性
3. **添加目录**：长文档建议添加目录
4. **代码高亮**：使用 ``` 指定语言类型
5. **图片优化**：使用图床或相对路径
