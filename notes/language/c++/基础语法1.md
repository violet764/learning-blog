# 变量与数据类型

本章介绍C++的变量、数据类型和运算符，是C++编程的基础内容。

## 概述

C++是一种静态类型的、编译式的、通用的、大小写敏感的编程语言，支持过程化编程、面向对象编程和泛型编程。

**核心概念：**
- **变量**：程序中用于存储数据的命名内存位置
- **数据类型**：定义变量可以存储的数据种类和大小
- **强类型语言**：变量必须声明类型，类型检查严格

### 第一个C++程序

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}
```

## 基础数据类型

C++原生支持的基础数据类型分为数值型、布尔型、空类型，尺寸受编译器和平台影响。

### 整数类型

整数类型用于存储整数值，分为有符号和无符号两种形式。

| 类型 | 典型尺寸 | 取值范围（典型） | 无符号形式 |
|------|----------|------------------|------------|
| `char` | 1字节 | -128 ~ 127 | `unsigned char` |
| `short` | 2字节 | -32768 ~ 32767 | `unsigned short` |
| `int` | 4字节 | $-2^{31}$ ~ $2^{31}-1$ | `unsigned int` |
| `long` | 4/8字节 | 平台相关 | `unsigned long` |
| `long long` | 8字节 | $-2^{63}$ ~ $2^{63}-1$ | `unsigned long long` |

**使用说明：**
- 无符号类型（`unsigned`）仅存储非负数，取值上限翻倍
- 字面量后缀：`U`（无符号）、`L`（long）、`LL`（long long）
- 用 `sizeof(类型)` 获取当前平台实际尺寸

```cpp
#include <iostream>
using namespace std;

int main() {
    // 整数类型声明
    int a = 100;
    unsigned int b = 100U;
    long long c = 1000000000LL;
    
    // 查看类型大小
    cout << "int大小: " << sizeof(int) << "字节" << endl;
    cout << "long long大小: " << sizeof(long long) << "字节" << endl;
    
    return 0;
}
```

### 浮点类型

浮点类型遵循IEEE 754标准，用于存储小数值，存在精度损失。

| 类型 | 典型尺寸 | 有效数字 | 取值范围 | 后缀 |
|------|----------|----------|----------|------|
| `float` | 4字节 | 6~7位 | ±$10^{-38}$ ~ ±$10^{38}$ | `f` |
| `double` | 8字节 | 15~16位 | ±$10^{-308}$ ~ ±$10^{308}$ | 无 |
| `long double` | 8/16字节 | 18~19位 | ±$10^{-4932}$ ~ ±$10^{4932}$ | `L` |

**注意事项：**
- 默认字面量为 `double` 类型
- 避免直接用 `==` 比较浮点数，需判断差值小于极小值

```cpp
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    float f = 3.14f;       // 单精度，需要f后缀
    double d = 3.14;       // 双精度，默认类型
    
    // 浮点数比较的正确方式
    double a = 0.1 + 0.2;
    if (fabs(a - 0.3) < 1e-6) {
        cout << "在容差范围内相等" << endl;
    }
    
    return 0;
}
```

### 布尔类型

布尔类型用于逻辑判断，只有 `true` 和 `false` 两个值。

| 类型 | 典型尺寸 | 取值 |
|------|----------|------|
| `bool` | 1字节 | `true`/`false` |

```cpp
bool is_valid = true;
bool is_empty = false;

// 非0值隐式转换为true，0转换为false
bool result = 5;      // true
result = 0;           // false
```

### 空类型

`void` 类型表示"无类型"，主要用于函数返回值和通用指针。

```cpp
// 无返回值的函数
void printHello() {
    cout << "Hello" << endl;
}

// 通用指针，可指向任意类型
void* ptr;
int a = 10;
ptr = &a;
```

## 复合数据类型

复合数据类型基于基本类型扩展，实现更复杂的数据存储逻辑。

### 指针

指针存储变量的内存地址，通过指针可以间接访问变量。

```cpp
int a = 10;
int* p = &a;       // p存储a的地址

cout << "a的值: " << a << endl;      // 10
cout << "a的地址: " << &a << endl;   // 地址
cout << "p的值: " << p << endl;      // 地址（与&a相同）
cout << "*p的值: " << *p << endl;    // 10（解引用）

// 空指针
int* null_ptr = nullptr;
```

**关键操作：**
- `&变量名`：取地址
- `*指针名`：解引用（访问指向的变量）
- `nullptr`：空指针常量（C++11）

### 引用

引用是变量的别名，必须初始化且不可更改指向。

```cpp
int a = 10;
int& ref = a;     // ref是a的别名

ref = 20;         // 修改ref就是修改a
cout << a << endl;  // 20

// 引用 vs 指针
// 引用：必须初始化，不可更改指向，无额外内存开销
// 指针：可以不初始化，可以更改指向，存储地址需要空间
```

### 数组

数组是相同类型数据的连续集合，下标从0开始。

```cpp
// 数组声明与初始化
int arr1[5];                       // 声明，未初始化
int arr2[5] = {1, 2, 3, 4, 5};     // 完整初始化
int arr3[5] = {1, 2};              // 部分初始化，剩余为0
int arr4[] = {1, 2, 3, 4, 5};      // 自动推断大小

// 访问元素
cout << arr2[0] << endl;           // 1

// 遍历数组
for (int i = 0; i < 5; i++) {
    cout << arr2[i] << " ";
}

// 范围for循环（C++11）
for (int num : arr2) {
    cout << num << " ";
}
```

### 字符串

C++支持两种字符串形式：C风格字符串和`std::string`。

**C风格字符串：**

以`\0`结尾的字符数组，继承自C语言。

```cpp
#include <cstring>

char str1[] = "Hello";           // 自动添加'\0'
char str2[20];
strcpy(str2, str1);              // 复制
strcat(str2, " World");          // 连接
cout << strlen(str2) << endl;    // 长度：11
```

**std::string：**

C++标准库的字符串类，自动管理内存，更加安全便捷。

```cpp
#include <string>

string s1 = "Hello";
string s2 = "World";
string s3 = s1 + " " + s2;        // 连接

cout << s3.length() << endl;     // 长度：11
cout << s3.substr(0, 5) << endl; // 子串：Hello
cout << s3.find("World") << endl; // 查找：6
```

## 自定义数据类型

自定义数据类型允许开发者封装新的数据结构。

### 结构体

结构体用于封装多个不同类型的变量。

```cpp
struct Student {
    string name;
    int age;
    float score;
    
    // 成员函数
    void show() {
        cout << name << ", " << age << "岁, 成绩: " << score << endl;
    }
};

// 使用
Student stu = {"张三", 20, 85.5};
stu.show();
```

### 枚举

枚举用于定义一组命名的整数常量。

**普通枚举：**

```cpp
enum Color {
    Red,      // 0
    Green,    // 1
    Blue      // 2
};

Color c = Red;
```

**强类型枚举（C++11）：**

解决普通枚举的作用域污染和类型安全问题。

```cpp
enum class Direction {
    Left,
    Right,
    Up,
    Down
};

Direction d = Direction::Left;  // 需要作用域限定
```

### 联合体

联合体的所有成员共享同一块内存空间，同一时间只能使用一个成员。

```cpp
union Data {
    int i;
    float f;
    char c;
};

Data d;
d.i = 10;
cout << d.i << endl;   // 10
d.f = 3.14f;           // 覆盖i的值
cout << d.f << endl;   // 3.14
```

## 变量声明与初始化

### 声明与初始化方式

```cpp
int x;              // 声明：内容未定义（危险）
int y = 10;         // 声明并初始化
int z{20};          // C++11统一初始化（推荐）

// 初始化方式的区别
int a = 3.14;       // 可能警告：窄化转换（3.14→3）
int b{3.14};        // 编译错误：阻止窄化转换

// 自动类型推导（C++11）
auto num = 42;              // 推导为int
auto name = "Alice";        // 推导为const char*
auto pi = 3.14;             // 推导为double
```

### 变量作用域

```cpp
int global_var = 100;   // 全局变量

int main() {
    int local_var = 200;    // 局部变量
    
    {
        int block_var = 300;  // 块作用域
        // 可以访问 global_var, local_var, block_var
    }
    
    // block_var 在此不可访问
    return 0;
}
```

## 常量

常量是程序运行期间不可修改的值。

### 常量定义方式

```cpp
// 方式1：#define预处理指令（C风格，不推荐）
#define MAX_SIZE 100

// 方式2：const关键字（推荐）
const int MIN_AGE = 18;
const double PI = 3.14159;

// 方式3：constexpr（C++11，编译时常量）
constexpr int ARRAY_SIZE = 50;
constexpr double E = 2.71828;
```

### const与constexpr的区别

```cpp
int runtime_value;
cin >> runtime_value;

const int const_value = runtime_value;     // 运行时常量，可以
// constexpr int ce_value = runtime_value;  // 错误：必须是编译时常量

constexpr int compile_time_value = 42;     // 编译时常量
```

### 常量指针与指针常量

const关键字的位置决定了指针的常量性质。

```cpp
int a = 10, b = 20;

// 1. 常量指针（const在*左边）：指向的值不能改
const int* p1 = &a;
// *p1 = 20;      // 错误：不能修改值
p1 = &b;          // 可以修改指向

// 2. 指针常量（const在*右边）：指向不能改
int* const p2 = &a;
*p2 = 20;         // 可以修改值
// p2 = &b;       // 错误：不能修改指向

// 3. 指向常量的指针常量：都不能改
const int* const p3 = &a;
// *p3 = 30;      // 错误
// p3 = &b;       // 错误

// 记忆技巧：const在*左边→值不变，const在*右边→指向不变
```

## 运算符与表达式

### 算术运算符

```cpp
int a = 10, b = 3;

cout << a + b << endl;   // 13
cout << a - b << endl;   // 7
cout << a * b << endl;   // 30
cout << a / b << endl;   // 3（整数除法，截断小数）
cout << a % b << endl;   // 1（取模，只能用于整数）

// 类型转换
double c = 10.0;
cout << c / b << endl;   // 3.333（double/int → double）
```

**注意事项：**
- 整数除法会截断小数部分
- 取模运算只能用于整数
- 不同类型运算时会自动类型提升

### 关系与逻辑运算符

```cpp
int x = 10, y = 5;

// 关系运算符
cout << (x == y) << endl;   // 0（false）
cout << (x != y) << endl;   // 1（true）
cout << (x > y) << endl;    // 1（true）

// 逻辑运算符
cout << (x > 0 && y > 0) << endl;  // 1（逻辑与）
cout << (x > 0 || y < 0) << endl;  // 1（逻辑或）
cout << !(x > 0) << endl;          // 0（逻辑非）

// 短路求值
// &&：第一个为假，第二个不执行
// ||：第一个为真，第二个不执行
```

### 赋值运算符

```cpp
int a = 10;
a += 5;   // a = a + 5 → 15
a -= 3;   // a = a - 3 → 12
a *= 2;   // a = a * 2 → 24
a /= 4;   // a = a / 4 → 6
a %= 4;   // a = a % 4 → 2
```

### 位运算符

位运算符直接操作整数的二进制位。

```cpp
unsigned int a = 0b1100;  // 12
unsigned int b = 0b1010;  // 10

cout << (a & b) << endl;   // 8  (1000) 按位与
cout << (a | b) << endl;   // 14 (1110) 按位或
cout << (a ^ b) << endl;   // 6  (0110) 按位异或
cout << (~a) << endl;      // 按位取反
cout << (a << 2) << endl;  // 48 (110000) 左移
cout << (a >> 2) << endl;  // 3  (0011) 右移
```

**位运算实用技巧：**

```cpp
// 检查第n位是否为1
bool isSet = (num & (1 << n)) != 0;

// 设置第n位为1
num |= (1 << n);

// 清除第n位
num &= ~(1 << n);

// 判断是否为2的幂
bool isPowerOfTwo = num && !(num & (num - 1));

// 交换两个变量（不用临时变量）
a ^= b; b ^= a; a ^= b;
```

## 类型转换

### 隐式类型转换

编译器自动进行的类型转换。

```cpp
int a = 10;
double b = a;    // int → double（类型提升）

char c = 'A';
int d = c;       // char → int（ASCII值）
```

### 显式类型转换

**C风格：**

```cpp
double price = 99.99;
int intPrice = (int)price;  // 截断小数部分
```

**C++风格（推荐）：**

```cpp
double f = 3.14;
int i = static_cast<int>(f);  // 更安全的转换

// 其他cast类型
// const_cast：移除const属性
// dynamic_cast：多态类型转换
// reinterpret_cast：底层重新解释
```

## 常见错误

### 整数溢出

```cpp
int big = 2000000000;
big = big * 2;  // 溢出！

// 修正：使用更大的类型
long long big = 2000000000LL;
big = big * 2;
```

### 浮点数精度问题

```cpp
double d1 = 0.1, d2 = 0.2;
if (d1 + d2 == 0.3) { /* 可能不成立 */ }

// 修正：使用容差比较
if (fabs(d1 + d2 - 0.3) < 1e-10) {
    cout << "在容差范围内相等" << endl;
}
```

### 未初始化变量

```cpp
int x;
cout << x << endl;  // 未定义行为

// 修正：始终初始化变量
int x = 0;
```

### 数组越界

```cpp
int arr[5] = {1, 2, 3, 4, 5};
cout << arr[5] << endl;  // 越界！

// 修正：确保下标在有效范围内
```
