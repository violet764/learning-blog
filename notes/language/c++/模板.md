# C++ 模板与泛型编程

模板是C++泛型编程的核心机制，允许编写与类型无关的通用代码。本章将详细介绍模板的基本概念、高级特性和最佳实践。

---

## 1. 模板基础

### 核心概念

**模板**是一种代码复用机制，通过类型参数化实现通用编程：
- **类型参数化**：将数据类型作为参数传递
- **编译时多态**：在编译时确定具体类型
- **代码复用**：同一模板可用于多种数据类型
- **性能优化**：避免运行时类型检查的开销

### 函数模板

```cpp
#include <iostream>
#include <string>
using namespace std;

// 基本函数模板：交换两个值
template <typename T>
void swapValues(T& a, T& b) {
    T temp = a;
    a = b;
    b = temp;
}

// 函数模板：查找最大值
template <typename T>
T findMax(const T arr[], int size) {
    T maxVal = arr[0];
    for (int i = 1; i < size; i++) {
        if (arr[i] > maxVal) {
            maxVal = arr[i];
        }
    }
    return maxVal;
}

// 多模板参数
template <typename T1, typename T2>
auto add(T1 a, T2 b) -> decltype(a + b) {
    return a + b;
}

int main() {
    // 使用模板函数交换整数
    int x = 10, y = 20;
    swapValues(x, y);
    cout << "交换后: x=" << x << ", y=" << y << endl;  // 20, 10
    
    // 交换字符串
    string s1 = "Hello", s2 = "World";
    swapValues(s1, s2);
    cout << "交换后: s1=" << s1 << ", s2=" << s2 << endl;
    
    // 查找数组最大值
    int intArr[] = {3, 7, 2, 9, 1};
    cout << "最大值: " << findMax(intArr, 5) << endl;  // 9
    
    // 多模板参数
    cout << "10 + 3.5 = " << add(10, 3.5) << endl;  // 13.5
    
    return 0;
}
```

### 类模板

```cpp
// 类模板：通用栈容器
template <typename T, int MAX_SIZE = 100>
class Stack {
private:
    T data[MAX_SIZE];
    int topIndex;
    
public:
    Stack() : topIndex(-1) {}
    
    bool push(const T& value) {
        if (topIndex < MAX_SIZE - 1) {
            data[++topIndex] = value;
            return true;
        }
        return false;
    }
    
    bool pop(T& value) {
        if (topIndex >= 0) {
            value = data[topIndex--];
            return true;
        }
        return false;
    }
    
    T top() const {
        if (topIndex >= 0) return data[topIndex];
        throw out_of_range("栈为空");
    }
    
    bool isEmpty() const { return topIndex == -1; }
    int size() const { return topIndex + 1; }
};

int main() {
    // 整数栈
    Stack<int> intStack;
    intStack.push(10);
    intStack.push(20);
    cout << "栈顶: " << intStack.top() << endl;  // 20
    
    // 字符串栈
    Stack<string> stringStack;
    stringStack.push("C++");
    stringStack.push("模板");
    
    string value;
    while (!stringStack.isEmpty()) {
        stringStack.pop(value);
        cout << value << " ";  // 模板 C++
    }
    cout << endl;
    
    // 自定义大小的栈
    Stack<double, 5> smallStack;
    for (int i = 0; i < 5; i++) {
        smallStack.push(i * 1.1);
    }
    
    return 0;
}
```

### 非类型模板参数

```cpp
// 非类型模板参数：编译时常量
template <typename T, int Size>
class Array {
private:
    T data[Size];
    
public:
    int size() const { return Size; }
    
    T& operator[](int index) {
        if (index < 0 || index >= Size) {
            throw out_of_range("索引越界");
        }
        return data[index];
    }
};

// 使用示例
Array<int, 5> arr;
arr[0] = 10;
cout << "数组大小: " << arr.size() << endl;  // 5
```

## 2. 模板特化与偏特化

### 完全特化

为特定类型提供专门实现：

```cpp
#include <iostream>
#include <string>
using namespace std;

// 通用模板
template <typename T>
class TypeInfo {
public:
    static string getName() { return "未知类型"; }
    static T zero() { return T(); }
};

// int 类型特化
template <>
class TypeInfo<int> {
public:
    static string getName() { return "整数"; }
    static int zero() { return 0; }
};

// double 类型特化
template <>
class TypeInfo<double> {
public:
    static string getName() { return "浮点数"; }
    static double zero() { return 0.0; }
};

// string 类型特化
template <>
class TypeInfo<string> {
public:
    static string getName() { return "字符串"; }
    static string zero() { return ""; }
};

int main() {
    cout << TypeInfo<int>::getName() << endl;      // 整数
    cout << TypeInfo<double>::getName() << endl;   // 浮点数
    cout << TypeInfo<string>::getName() << endl;   // 字符串
    cout << TypeInfo<char>::getName() << endl;     // 未知类型
    return 0;
}
```

### 偏特化

为部分类型参数提供特殊实现：

```cpp
// 通用模板
template <typename T>
class Container {
public:
    void process(T value) {
        cout << "通用处理: " << value << endl;
    }
};

// 指针类型的偏特化
template <typename T>
class Container<T*> {
public:
    void process(T* ptr) {
        cout << "指针处理: ";
        if (ptr) cout << *ptr << endl;
        else cout << "nullptr" << endl;
    }
};

// 数组类型的偏特化
template <typename T, int N>
class Container<T[N]> {
public:
    void process(T arr[N]) {
        cout << "数组处理: ";
        for (int i = 0; i < N; i++) {
            cout << arr[i] << " ";
        }
        cout << endl;
    }
};

int main() {
    Container<int> c1;
    c1.process(42);  // 通用处理: 42
    
    Container<int*> c2;
    int x = 100;
    c2.process(&x);  // 指针处理: 100
    
    Container<int[3]> c3;
    int arr[] = {1, 2, 3};
    c3.process(arr);  // 数组处理: 1 2 3
    
    return 0;
}
```

## 3. 可变参数模板

### 基本语法

```cpp
#include <iostream>
using namespace std;

// 递归终止条件
void print() {
    cout << endl;
}

// 可变参数模板
template <typename T, typename... Args>
void print(T first, Args... rest) {
    cout << first;
    if (sizeof...(rest) > 0) cout << ", ";
    print(rest...);  // 递归展开
}

int main() {
    print(1, 2.0, "three", '4');
    // 输出: 1, 2, three, 4
    return 0;
}
```

### 折叠表达式（C++17）

```cpp
// 一元右折叠
template <typename... Args>
auto sum(Args... args) {
    return (args + ...);  // 等价于: arg1 + (arg2 + (arg3 + ...))
}

// 一元左折叠
template <typename... Args>
auto product(Args... args) {
    return (... * args);  // 等价于: ((arg1 * arg2) * arg3) * ...
}

// 二元折叠（带初始值）
template <typename... Args>
auto sumWithInit(Args... args) {
    return (0 + ... + args);  // 初始值为0
}

int main() {
    cout << "sum: " << sum(1, 2, 3, 4, 5) << endl;        // 15
    cout << "product: " << product(1, 2, 3, 4) << endl;   // 24
    cout << "sumWithInit: " << sumWithInit(1, 2, 3) << endl;  // 6
    return 0;
}
```

### 完美转发

```cpp
#include <utility>
#include <string>

// 完美转发：保持参数的值类别
template <typename T>
void wrapper(T&& arg) {
    process(std::forward<T>(arg));
}

// 实际应用：工厂函数
template <typename T, typename... Args>
std::unique_ptr<T> make(Args&&... args) {
    return std::make_unique<T>(std::forward<Args>(args)...);
}
```

## 4. 模板元编程

### 编译时计算

```cpp
// 编译时计算阶乘
template <int N>
struct Factorial {
    static constexpr int value = N * Factorial<N - 1>::value;
};

template <>
struct Factorial<0> {
    static constexpr int value = 1;
};

// 编译时斐波那契数列
template <int N>
struct Fibonacci {
    static constexpr int value = Fibonacci<N-1>::value + Fibonacci<N-2>::value;
};

template <>
struct Fibonacci<0> { static constexpr int value = 0; };

template <>
struct Fibonacci<1> { static constexpr int value = 1; };

int main() {
    cout << "5! = " << Factorial<5>::value << endl;    // 120
    cout << "F(10) = " << Fibonacci<10>::value << endl;  // 55
    return 0;
}
```

### 类型特性检测

```cpp
#include <type_traits>

// 使用SFINAE检测类型特性
template <typename T, typename = void>
struct has_size : std::false_type {};

template <typename T>
struct has_size<T, std::void_t<decltype(std::declval<T>().size())>> 
    : std::true_type {};

// 使用if constexpr（C++17）
template <typename T>
void process(T value) {
    if constexpr (std::is_integral_v<T>) {
        cout << "整数: " << value << endl;
    } else if constexpr (std::is_floating_point_v<T>) {
        cout << "浮点数: " << value << endl;
    } else if constexpr (has_size<T>::value) {
        cout << "容器大小: " << value.size() << endl;
    } else {
        cout << "其他类型" << endl;
    }
}
```

## 5. 智能指针与内存管理

### 智能指针概述

```cpp
#include <memory>
#include <iostream>

class Resource {
public:
    std::string name;
    Resource(const std::string& n) : name(n) {
        std::cout << "创建: " << name << std::endl;
    }
    ~Resource() {
        std::cout << "销毁: " << name << std::endl;
    }
};

void smartPointerDemo() {
    // unique_ptr：独占所有权
    auto unique = std::make_unique<Resource>("unique");
    auto moved = std::move(unique);  // 转移所有权
    
    // shared_ptr：共享所有权
    auto shared1 = std::make_shared<Resource>("shared");
    auto shared2 = shared1;  // 共享所有权
    std::cout << "引用计数: " << shared1.use_count() << std::endl;  // 2
    
    // weak_ptr：不增加引用计数
    std::weak_ptr<Resource> weak = shared1;
    if (auto locked = weak.lock()) {
        std::cout << "访问: " << locked->name << std::endl;
    }
}
```

### 循环引用问题

```cpp
class Node {
public:
    std::string name;
    std::shared_ptr<Node> next;
    std::weak_ptr<Node> prev;  // 使用weak_ptr避免循环引用
    
    Node(const std::string& n) : name(n) {}
    ~Node() { std::cout << "销毁节点: " << name << std::endl; }
};

void circularRefDemo() {
    auto node1 = std::make_shared<Node>("A");
    auto node2 = std::make_shared<Node>("B");
    
    node1->next = node2;
    node2->prev = node1;  // weak_ptr不增加引用计数
    
    std::cout << "node1引用计数: " << node1.use_count() << std::endl;  // 1
    // 离开作用域时两个节点都能正确销毁
}
```

### 自定义删除器

```cpp
#include <cstdio>

void customDeleterDemo() {
```

## 6. RAII与异常安全

### RAII原则

**RAII（Resource Acquisition Is Initialization）**：资源获取即初始化，利用对象生命周期管理资源。

```cpp
#include <fstream>
#include <stdexcept>

class FileHandler {
private:
    std::unique_ptr<std::fstream> file;
    
public:
    FileHandler(const std::string& filename) {
        file = std::make_unique<std::fstream>(filename);
        if (!file->is_open()) {
            throw std::runtime_error("无法打开文件");
        }
    }
    
    void write(const std::string& data) {
        *file << data;
    }
    
    // 析构函数自动关闭文件（RAII）
};

void safeFileOperation() {
    FileHandler handler("data.txt");
    handler.write("Hello, RAII!");
    // 离开作用域自动关闭文件，即使抛出异常也能正确释放
}
```

### 异常安全保证

```cpp
class SafeContainer {
private:
    std::unique_ptr<int[]> data;
    size_t size;
    
public:
    // 基本保证：不泄漏资源
    void resize(size_t newSize) {
        auto newData = std::make_unique<int[]>(newSize);
        // 复制数据...
        data = std::move(newData);  // 无异常操作
        size = newSize;
    }
    
    // 强保证：操作具有原子性
    void safeUpdate(int value) {
        auto oldData = std::make_unique<int[]>(size);
        // 备份旧数据...
        
        try {
            // 可能抛出异常的操作
            if (value < 0) throw std::invalid_argument("负值");
            // 更新数据...
        } catch (...) {
            data = std::move(oldData);  // 回滚
            throw;
        }
    }
    
    // 不抛出保证
    int getValue(size_t index) const noexcept {
        return (index < size) ? data[index] : 0;
    }
};
```

## 7. 常见错误与最佳实践

### 易错点

#### 1. 模板定义分离

```cpp
// 错误：模板定义放在源文件中
// mytemplate.h
template <typename T>
T add(T a, T b);  // 只有声明

// mytemplate.cpp
template <typename T>
T add(T a, T b) { return a + b; }  // 定义

// main.cpp
#include "mytemplate.h"
int result = add(1, 2);  // 链接错误！

// 正确：模板定义放在头文件中
template <typename T>
T add(T a, T b) { return a + b; }
```

#### 2. 类型要求不满足

```cpp
template <typename T>
T getAverage(const T arr[], int size) {
    T sum = T();
    for (int i = 0; i < size; i++) sum += arr[i];
    return sum / size;  // 要求T支持+=和/操作
}

class MyClass { /* 没有+=和/运算符 */ };

MyClass arr[3];
getAverage(arr, 3);  // 编译错误！

// 正确：使用概念约束（C++20）
template <typename T>
requires std::is_arithmetic_v<T>
T getAverage(const T arr[], int size) { /* ... */ }
```

#### 3. 内存泄漏与悬空指针

```cpp
// 错误：忘记释放内存
void memoryLeak() {
    int* ptr = new int[100];
    // 使用ptr...
    // 忘记delete[] ptr;  // 内存泄漏！
}

// 正确：使用智能指针
void safeMemory() {
    auto ptr = std::make_unique<int[]>(100);
    // 自动释放
}
```

### 最佳实践

1. **模板定义放头文件**：确保模板定义对使用者可见
2. **使用概念约束**（C++20）：明确模板参数要求
3. **优先使用智能指针**：避免手动内存管理
4. **遵循RAII原则**：资源管理与对象生命周期绑定
5. **提供异常安全保证**：确保异常发生时资源正确释放

## 8. 现代C++特性

### 概念（Concepts，C++20）

```cpp
#include <concepts>

// 定义概念
template <typename T>
concept Numeric = std::is_arithmetic_v<T>;

template <typename T>
concept Printable = requires(std::ostream& os, T value) {
    { os << value } -> std::same_as<std::ostream&>;
};

// 使用概念约束
template <Numeric T>
T add(T a, T b) { return a + b; }

template <Printable T>
void print(const T& value) {
    std::cout << value << std::endl;
}
```

### 结构化绑定（C++17）

```cpp
#include <map>
#include <tuple>

void structuredBindings() {
    std::map<std::string, int> scores = {{"Alice", 90}, {"Bob", 85}};
    
    for (const auto& [name, score] : scores) {
        std::cout << name << ": " << score << std::endl;
    }
    
    auto [x, y, z] = std::make_tuple(1, 2.0, "three");
    std::cout << x << ", " << y << ", " << z << std::endl;
}
```

### if constexpr（C++17）

```cpp
template <typename T>
auto getValue(T t) {
    if constexpr (std::is_pointer_v<T>) {
        return *t;  // 解引用
    } else {
        return t;   // 直接返回
    }
}
```
