# 面向对象编程基础

面向对象编程（Object-Oriented Programming, OOP）是一种以对象为核心的编程范式。C++ 作为一门支持多范式的语言，提供了完整的面向对象特性支持。本文将介绍 C++ 面向对象编程的基础概念和核心机制。

## 基本概念

### 核心定义

| 概念 | 说明 |
|------|------|
| **类（Class）** | 创建对象的蓝图或模板，定义了对象的属性和行为 |
| **对象（Object）** | 类的实例，具有状态和行为的实体 |
| **封装（Encapsulation）** | 将数据和操作数据的方法捆绑在一起，隐藏实现细节 |
| **继承（Inheritance）** | 基于已有类创建新类，实现代码复用 |
| **多态（Polymorphism）** | 同一接口，不同实现，运行时动态绑定 |

### 面向对象的四大特性

```
┌─────────────────────────────────────────────────────────────┐
│                      面向对象四大特性                         │
├─────────────┬─────────────┬─────────────┬─────────────────────┤
│    抽象     │    封装     │    继承     │       多态          │
├─────────────┼─────────────┼─────────────┼─────────────────────┤
│ 隐藏复杂实现 │ 数据保护    │ 代码复用    │ 同一接口不同实现     │
│ 只暴露必要   │ 控制访问权限 │ 建立类关系  │ 运行时动态绑定       │
└─────────────┴─────────────┴─────────────┴─────────────────────┘
```

## 类与对象

### 类的基本定义

```cpp
class 类名 {
访问修饰符:
    // 数据成员（属性）
    数据类型 成员变量;
    
    // 成员函数（方法）
    返回类型 函数名(参数列表);
};  // ⚠️ 注意：类定义必须以分号结尾
```

### 基础示例：Person 类

```cpp
#include <iostream>
#include <string>
using namespace std;

class Person {
public:  // 公有访问权限
    // 数据成员
    string name;
    int age;
    
    // 成员函数
    void introduce() {
        cout << "我叫" << name << ", 今年" << age << "岁。" << endl;
    }
    
    void setAge(int newAge) {
        if (newAge > 0 && newAge < 150) {
            age = newAge;
        }
    }
};

int main() {
    // 创建对象
    Person person1;
    
    // 访问公有成员
    person1.name = "张三";
    person1.setAge(25);
    
    // 调用成员函数
    person1.introduce();  // 输出：我叫张三, 今年25岁。
    
    return 0;
}
```

### 访问修饰符

C++ 提供三种访问修饰符来控制成员的可访问性：

```cpp
class AccessDemo {
private:    // 🔒 私有成员：仅类内部可访问
    int privateVar;
    
protected:  // 🛡️ 保护成员：类内部和派生类可访问
    int protectedVar;
    
public:     // 🔓 公有成员：任何地方都可访问
    int publicVar;
};
```

| 访问修饰符 | 类内部 | 派生类 | 外部代码 |
|-----------|--------|--------|----------|
| `private` | ✅ | ❌ | ❌ |
| `protected` | ✅ | ✅ | ❌ |
| `public` | ✅ | ✅ | ✅ |

## 封装性

### 封装的意义

封装是面向对象编程的核心特性之一，它将数据和对数据的操作捆绑在一起，通过访问控制实现信息隐藏。

**封装的优点：**
- 📌 **数据保护**：防止外部代码意外修改内部状态
- 📌 **接口清晰**：只暴露必要的公有接口
- 📌 **实现隐藏**：内部实现可以自由修改而不影响使用者
- 📌 **维护性强**：降低代码耦合度

### 封装示例：银行账户类

```cpp
#include <iostream>
#include <string>
using namespace std;

class BankAccount {
private:    // 私有数据成员，外部不能直接访问
    string accountNumber;
    string ownerName;
    double balance;
    
public:     // 公有接口，外部可以访问
    // 构造函数
    BankAccount(string accNum, string name, double initialBalance) {
        accountNumber = accNum;
        ownerName = name;
        balance = (initialBalance >= 0) ? initialBalance : 0;
    }
    
    // 存款方法
    bool deposit(double amount) {
        if (amount > 0) {
            balance += amount;
            cout << "存款成功，当前余额: " << balance << endl;
            return true;
        }
        cout << "存款金额必须大于0" << endl;
        return false;
    }
    
    // 取款方法
    bool withdraw(double amount) {
        if (amount > 0 && amount <= balance) {
            balance -= amount;
            cout << "取款成功，当前余额: " << balance << endl;
            return true;
        }
        cout << "取款失败，余额不足或金额无效" << endl;
        return false;
    }
    
    // 获取余额（只读访问）
    double getBalance() const {
        return balance;
    }
    
    // 获取账户信息
    void displayInfo() const {
        cout << "账户号: " << accountNumber << endl;
        cout << "户主: " << ownerName << endl;
        cout << "余额: " << balance << endl;
    }
};

int main() {
    BankAccount myAccount("123456789", "张三", 1000.0);
    
    myAccount.displayInfo();
    myAccount.deposit(500.0);
    myAccount.withdraw(200.0);
    
    // 无法直接访问私有成员
    // myAccount.balance = 10000;  // ❌ 错误：私有成员不能访问
    
    cout << "当前余额: " << myAccount.getBalance() << endl;
    return 0;
}
```

## 构造函数与析构函数

### 构造函数

构造函数在对象创建时自动调用，负责初始化对象状态。C++ 提供多种构造函数类型：

```cpp
#include <iostream>
#include <string>
using namespace std;

class Person {
private:
    string name;
    int age;
    
public:
    // 1️⃣ 默认构造函数：无参初始化
    Person() : name("Unknown"), age(0) {
        cout << "默认构造函数调用" << endl;
    }
    
    // 2️⃣ 参数化构造函数：带参数的初始化
    Person(const string &n, int a) : name(n), age(a) {
        cout << "参数化构造函数调用" << endl;
    }
    
    // 3️⃣ 委托构造函数（C++11）：重用其他构造函数
    Person(const string &n) : Person(n, 0) {
        cout << "委托构造函数调用" << endl;
    }
    
    // 4️⃣ 拷贝构造函数：创建对象的副本
    Person(const Person &other) : name(other.name), age(other.age) {
        cout << "拷贝构造函数调用" << endl;
    }
    
    // 5️⃣ 移动构造函数（C++11）：资源转移
    Person(Person &&other) noexcept 
        : name(std::move(other.name)), age(other.age) {
        other.age = 0;
        cout << "移动构造函数调用" << endl;
    }
    
    // 析构函数
    ~Person() {
        cout << name << " 对象被销毁" << endl;
    }
    
    void display() const {
        cout << "姓名: " << name << ", 年龄: " << age << endl;
    }
};

int main() {
    Person p1;                     // 默认构造
    Person p2("Alice", 25);        // 参数化构造
    Person p3 = p2;                // 拷贝构造
    Person p4 = std::move(p3);     // 移动构造
    Person p5{"Bob"};              // 统一初始化语法
    
    return 0;
}
```

### 成员初始化列表

成员初始化列表在对象内存分配后立即执行，直接构造成员，比在构造函数体内赋值更高效：

```cpp
class Student {
private:
    const int id;           // const成员必须使用初始化列表
    string name;
    int& ageRef;            // 引用成员必须使用初始化列表
    
public:
    // ✅ 推荐方式：使用初始化列表
    Student(int studentId, const string& studentName, int& age) 
        : id(studentId), name(studentName), ageRef(age) {
        // 初始化列表已经完成了所有成员的构造
    }
    
    // ❌ 不推荐：在构造函数体内赋值（效率较低）
    // Student(int studentId, string studentName) {
    //     id = studentId;     // 错误！const成员不能赋值
    //     name = studentName; // 先默认构造，再赋值，效率低
    // }
};
```

**必须使用初始化列表的情况：**
- const 成员变量
- 引用成员变量
- 没有默认构造函数的类类型成员

### 析构函数

析构函数在对象销毁时自动调用，负责释放资源：

```cpp
class Resource {
private:
    string name;
    int* data;
    
public:
    Resource(const string& n) : name(n), data(new int[100]) {
        cout << name << ": 资源已分配" << endl;
    }
    
    // 析构函数：释放资源
    ~Resource() {
        delete[] data;  // 释放动态分配的内存
        cout << name << ": 资源已释放" << endl;
    }
};

void test() {
    Resource r1("资源1");
    {
        Resource r2("资源2");
        // r2 在此作用域结束时销毁
    }  // r2 析构
    // r1 在函数结束时销毁
}  // r1 析构
```

## this 指针

每个非静态成员函数都隐含一个 `this` 指针参数，指向调用该函数的对象：

```cpp
class Point {
private:
    int x, y;
    
public:
    // 使用 this 解决命名冲突
    Point(int x, int y) {
        this->x = x;  // this->x 是成员变量，x 是参数
        this->y = y;
    }
    
    // 链式调用：返回对象引用支持连续操作
    Point& setX(int x) {
        this->x = x;
        return *this;  // 返回当前对象的引用
    }
    
    Point& setY(int y) {
        this->y = y;
        return *this;
    }
    
    void display() const {
        cout << "Point(" << x << ", " << y << ")" << endl;
    }
};

int main() {
    Point p(10, 20);
    p.setX(30).setY(40).display();  // 链式调用
    return 0;
}
```

## 静态成员

静态成员属于类本身而非对象实例，所有对象共享同一份静态成员：

```cpp
class Counter {
private:
    static int count;  // 静态成员变量：所有对象共享
    int id;            // 普通成员变量：每个对象独立
    
public:
    Counter() : id(++count) {}
    
    // 静态成员函数：属于类而非对象
    static int getCount() {
        return count;  // 只能访问静态成员
    }
    
    int getId() const { return id; }
    
    ~Counter() { --count; }
};

// 静态成员变量定义：必须在类外定义
int Counter::count = 0;

int main() {
    Counter c1, c2, c3;
    cout << "对象数量: " << Counter::getCount() << endl;  // 3
    cout << "c1 的 ID: " << c1.getId() << endl;           // 1
    cout << "c2 的 ID: " << c2.getId() << endl;           // 2
    return 0;
}
```

### 静态成员的特性

| 特性 | 说明 |
|------|------|
| **存储位置** | 程序的数据段，生命周期与程序相同 |
| **共享性** | 所有对象共享同一份静态成员 |
| **访问方式** | `类名::静态成员` 或 `对象.静态成员` |
| **初始化** | 必须在类外定义（C++17 可用 `inline static`） |

## const 成员函数

const 成员函数承诺不修改对象状态：

```cpp
class Student {
private:
    string name;
    mutable int accessCount;  // mutable 可在 const 函数中修改
    
public:
    Student(const string& n) : name(n), accessCount(0) {}
    
    // const 成员函数：承诺不修改对象状态
    const string& getName() const {
        accessCount++;  // 可以修改 mutable 成员
        return name;
    }
    
    // 非 const 版本
    void setName(const string& n) {
        name = n;
    }
};

int main() {
    Student s("张三");
    const Student& constRef = s;
    
    constRef.getName();     // ✅ const 对象可调用 const 成员函数
    // constRef.setName("李四");  // ❌ const 对象不能调用非 const 成员函数
    
    return 0;
}
```

## 常见错误

### 1. 忘记类定义的分号

```cpp
class MyClass {
    // 类内容
}  // ❌ 错误：缺少分号

class CorrectClass {
    // 类内容
};  // ✅ 正确：添加分号
```

### 2. 直接访问私有成员

```cpp
class MyClass {
private:
    int secret;
};

int main() {
    MyClass obj;
    // obj.secret = 10;  // ❌ 错误：私有成员不能直接访问
    return 0;
}
```

### 3. const 成员函数修改数据

```cpp
class ConstError {
private:
    int value;
    
public:
    int getValue() const {
        // value = 10;  // ❌ 错误：const 函数不能修改非 mutable 成员
        return value;
    }
};
```

### 4. 违反封装原则

```cpp
// ❌ 错误示例：所有成员都是公有的
class BadAccount {
public:
    double balance;  // 外部可以直接修改，破坏数据完整性
};

// ✅ 正确示例：良好的封装
class GoodAccount {
private:
    double balance;
    
public:
    void withdraw(double amount) {
        if (amount > 0 && amount <= balance) {
            balance -= amount;
        }
    }
    
    double getBalance() const { return balance; }
};
```
