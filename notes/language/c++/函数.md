# 函数

本章介绍C++函数的声明、定义、参数传递、重载、内联和递归等核心概念，是C++模块化编程的基础。

## 概述

函数是完成特定任务的独立代码块，通过函数可以实现代码复用、模块化设计和逻辑分离。每个C++程序都至少有一个函数`main()`，它是程序的入口点。

**核心概念：**
- **函数声明**：告诉编译器函数的存在
- **函数定义**：提供函数的具体实现
- **参数传递**：数据传入函数的方式
- **函数重载**：同名函数的多种实现

---

## 函数声明与定义

### 函数基本结构

```cpp
// 函数定义形式
返回类型 函数名(参数列表) {
    // 函数体
    return 返回值;
}

// 示例：两数相加
int add(int a, int b) {
    return a + b;
}
```

### 声明与定义分离

通常将函数声明放在头文件中，函数定义放在源文件中，实现接口与实现的分离。

```cpp
// math_utils.h（头文件）
#ifndef MATH_UTILS_H
#define MATH_UTILS_H

// 函数声明
int multiply(int x, int y);
double power(double base, int exponent);

#endif

// math_utils.cpp（源文件）
#include "math_utils.h"

// 函数定义
int multiply(int x, int y) {
    return x * y;
}

double power(double base, int exponent) {
    double result = 1.0;
    for (int i = 0; i < exponent; i++) {
        result *= base;
    }
    return result;
}
```

## 参数传递机制

C++支持三种参数传递方式：值传递、引用传递和指针传递，不同的传递方式有不同的语义和适用场景。

### 值传递

值传递会复制实参的值给形参，函数内对形参的修改不影响原变量。

```cpp
void modifyValue(int x) {  // x是a的副本
    x = 100;  // 只修改副本，不影响原变量
}

int main() {
    int a = 10;
    modifyValue(a);
    cout << a << endl;  // 输出: 10（原值未改变）
    return 0;
}
```

**适用场景**：基本类型、小型对象、不需要修改原变量时。

### 引用传递

引用传递将形参作为实参的别名，函数内对形参的修改直接影响原变量，且不会产生数据拷贝。

```cpp
void modifyReference(int &x) {  // x是a的引用
    x = 100;  // 直接修改原变量
}

// 常量引用：只读访问，避免拷贝
void readOnly(const vector<int> &vec) {
    cout << "Size: " << vec.size() << endl;
}

int main() {
    int a = 10;
    modifyReference(a);
    cout << a << endl;  // 输出: 100
    return 0;
}
```

**适用场景**：需要修改原变量、传递大型对象。

### 指针传递

指针传递将变量的地址传给函数，通过解引用可以访问和修改原变量。

```cpp
void modifyPointer(int *x) {
    *x = 100;  // 通过指针修改原变量
}

int main() {
    int a = 10;
    modifyPointer(&a);
    cout << a << endl;  // 输出: 100
    return 0;
}
```

**适用场景**：需要修改原变量、需要处理空指针情况、C风格兼容。

### 三种传递方式对比

| 传递方式 | 内存开销 | 是否修改原变量 | 适用场景 |
|----------|----------|----------------|----------|
| **值传递** | 创建完整副本 | 否 | 基本类型、小型对象 |
| **引用传递** | 仅传递别名 | 是 | 需要修改原变量、大型对象 |
| **常量引用** | 仅传递别名 | 否 | 只读访问大型对象 |
| **指针传递** | 传递地址值 | 是 | 需要修改原变量、可空参数 |

## 函数重载

函数重载允许定义多个同名函数，只要它们的参数列表不同即可。编译器会根据调用时的实参类型自动选择正确的版本。

### 重载示例

```cpp
// 重载：同名不同参数
int add(int a, int b) {
    return a + b;
}

double add(double a, double b) {
    return a + b;
}

string add(const string &a, const string &b) {
    return a + b;
}

// 调用时编译器自动选择
cout << add(5, 3) << endl;               // int版本
cout << add(2.5, 3.7) << endl;           // double版本
cout << add("Hello", " World") << endl;  // string版本
```

### 重载规则

1. 参数类型或数量必须不同
2. 返回值类型不能作为重载依据
3. const成员函数与非const成员函数可以重载
4. 默认参数可能引起重载歧义

## 默认参数

默认参数允许在函数声明中为参数指定默认值，调用时可以省略这些参数。

```cpp
// 默认参数必须从右向左定义
int calculateArea(int length, int width = 1) {
    return length * width;
}

void printMessage(string message, int times = 1) {
    for (int i = 0; i < times; i++) {
        cout << message << endl;
    }
}

int main() {
    cout << calculateArea(5) << endl;      // 5 (使用默认宽度)
    cout << calculateArea(5, 3) << endl;   // 15
    
    printMessage("Hello");      // 打印1次
    printMessage("World", 3);   // 打印3次
    
    return 0;
}
```

**默认参数规则：**
- 默认参数必须从右向左定义
- 默认参数在函数声明中指定（通常在头文件中）
- 调用时可以省略有默认值的参数

## 内联函数

内联函数是一种编译优化手段，编译器会将函数体代码直接嵌入调用处，减少函数调用的开销。

### inline函数

```cpp
// 建议编译器将函数体直接插入调用处
inline int square(int x) {
    return x * x;
}

int main() {
    int result = square(5);  // 可能被优化为: 5 * 5
    return 0;
}
```

**适用场景**：小型、频繁调用的函数。

### constexpr函数

constexpr函数可以在编译时计算，提高运行时性能。

```cpp
constexpr int factorial(int n) {
    return (n <= 1) ? 1 : n * factorial(n - 1);
}

int main() {
    constexpr int fact_5 = factorial(5);  // 编译时计算为120
    return 0;
}
```

## 递归函数

递归函数是指在函数体内调用自身的函数。递归必须有一个终止条件，否则会导致无限递归。

### 基本示例

```cpp
// 递归计算阶乘
int factorial(int n) {
    if (n <= 0) {  // 递归终止条件
        return 1;
    }
    return n * factorial(n - 1);  // 递归步骤
}

int main() {
    cout << "5! = " << factorial(5) << endl;  // 120
    return 0;
}
```

### 递归过程分析

```
factorial(5)
= 5 × factorial(4)
= 5 × 4 × factorial(3)
= 5 × 4 × 3 × factorial(2)
= 5 × 4 × 3 × 2 × factorial(1)
= 5 × 4 × 3 × 2 × 1 × factorial(0)
= 5 × 4 × 3 × 2 × 1 × 1
= 120
```

### 递归与迭代对比

```cpp
// 递归版本
int factorial_recursive(int n) {
    if (n <= 0) return 1;
    return n * factorial_recursive(n - 1);
}

// 迭代版本
int factorial_iterative(int n) {
    int result = 1;
    for (int i = 1; i <= n; i++) {
        result *= i;
    }
    return result;
}
```

**注意事项**：
- 递归代码简洁但可能有栈溢出风险
- 迭代版本通常性能更好
- 尾递归可被编译器优化为迭代

## 函数指针

函数指针是指向函数的指针变量，可以用于回调函数、策略模式等场景。

### 基本用法

```cpp
// 定义函数指针
int (*funcPtr)(int, int);

// 指向具体函数
funcPtr = add;

// 通过函数指针调用
int result = funcPtr(5, 3);  // 等价于 add(5, 3)

// 使用typedef简化
typedef int (*MathFunc)(int, int);
MathFunc operation = add;
```

### 回调函数示例

```cpp
// 接受函数指针作为参数
void processArray(int arr[], int size, int (*operation)(int)) {
    for (int i = 0; i < size; i++) {
        arr[i] = operation(arr[i]);
    }
}

int square(int x) { return x * x; }
int doubleValue(int x) { return x * 2; }

int main() {
    int arr[] = {1, 2, 3, 4, 5};
    
    processArray(arr, 5, square);      // 每个元素平方
    processArray(arr, 5, doubleValue); // 每个元素翻倍
    
    return 0;
}
```

## Lambda表达式

Lambda表达式是C++11引入的匿名函数，可以方便地定义内联函数对象。

### 基本语法

```cpp
[capture](parameters) -> return_type { function_body }
```

### 示例

```cpp
#include <algorithm>
#include <vector>

int main() {
    // 简单lambda
    auto greet = []() { cout << "Hello!" << endl; };
    greet();
    
    // 带参数的lambda
    auto add = [](int a, int b) { return a + b; };
    cout << add(5, 3) << endl;  // 8
    
    // 捕获外部变量
    int x = 10;
    auto addX = [x](int n) { return n + x; };  // 值捕获
    auto addXRef = [&x](int n) { return n + x; };  // 引用捕获
    
    // 与算法配合使用
    vector<int> nums = {1, 2, 3, 4, 5};
    for_each(nums.begin(), nums.end(), [](int n) {
        cout << n << " ";
    });
    
    return 0;
}
```

### 捕获方式

| 捕获方式 | 说明 |
|----------|------|
| `[]` | 不捕获任何变量 |
| `[=]` | 值捕获所有外部变量 |
| `[&]` | 引用捕获所有外部变量 |
| `[x]` | 值捕获变量x |
| `[&x]` | 引用捕获变量x |
| `[=, &x]` | 值捕获所有，引用捕获x |
| `[&, x]` | 引用捕获所有，值捕获x |

## 常见错误

### 值传递无法修改原变量

```cpp
// 错误示例
void swap(int a, int b) {  // 值传递
    int temp = a;
    a = b;
    b = temp;
}

// 正确做法：使用引用传递
void swap(int &a, int &b) {
    int temp = a;
    a = b;
    b = temp;
}
```

### 无限递归

```cpp
// 错误示例：缺少终止条件
int badRecursion(int n) {
    return n + badRecursion(n - 1);  // 无限递归
}

// 正确做法
int goodRecursion(int n) {
    if (n <= 0) {  // 终止条件
        return 0;
    }
    return n + goodRecursion(n - 1);
}
```

### 返回局部变量的引用

```cpp
// 错误示例：返回局部变量的引用
int& badFunction() {
    int local = 10;
    return local;  // 返回局部变量的引用，悬空引用
}

// 正确做法：返回值或静态变量
int goodFunction() {
    int local = 10;
    return local;  // 返回值，安全
}
```
