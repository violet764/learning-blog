# IDE 介绍


## VS Code 常用快捷键

|功能描述|Windows|
|---|---|
|打开命令面板（万能入口）|`Ctrl + Shift + P` / `F1`|
|快速打开文件|`Ctrl + P`|
|保存文件|`Ctrl + S`|
|撤销|`Ctrl + Z`|
|重做|`Ctrl + Y`|
|复制|`Ctrl + C`|
|剪切|`Ctrl + X`|
|粘贴|`Ctrl + V`|
|全选|`Ctrl + A`|
|查找内容|`Ctrl + F`|
|替换内容|`Ctrl + H`|
|注释/取消注释（单行）|`Ctrl + /`|
|格式化代码|`Shift + Alt + F`|
|移动当前行（上下）|`Alt + Up/Down`|
|复制当前行（上下）|`Shift + Alt + Up/Down`|
|删除当前行|`Ctrl + Shift + K`|
|打开/隐藏终端|``Ctrl + ` ``（反引号）|
|打开/隐藏侧边栏|`Ctrl + B`|
|跳转到行号|`Ctrl + G`|
|分栏编辑（垂直）|`Ctrl + \`|
|切换标签页|`Ctrl + Tab`|
|关闭当前标签页|`Ctrl + W`|
|跳转到定义|`F12`|
|多光标添加（点击）|`Alt + 鼠标点击`|


## Pycharm 常用快捷键


|功能描述|Windows 快捷键|
|---|---|
|保存文件|`Ctrl + S`|
|撤销|`Ctrl + Z`|
|重做|`Ctrl + Shift + Z`|
|复制|`Ctrl + C`|
|剪切|`Ctrl + X`|
|粘贴|`Ctrl + V`|
|单行注释/取消注释|`Ctrl + /`|
|块注释/取消注释|`Ctrl + Shift + /`|
|格式化代码|`Ctrl + Alt + L`|
|移动当前行（上下）|`Alt + Shift + Up/Down`|
|复制当前行（上下）|`Ctrl + D`|
|删除当前行|`Ctrl + Y`|
|自动补全代码|`Ctrl + Space`|
|查找内容|`Ctrl + F`|
|替换内容|`Ctrl + R`|
|全局搜索（文件/内容）|双击Shift|
|跳转到定义|`Ctrl + B` / `Ctrl + 鼠标点击`|
|跳转到行号|`Ctrl + G`|
|切换标签页|`Ctrl + Tab`|
|开始/重启调试|`Shift + F9`|
|停止调试|`Ctrl + F2`|
|单步执行（不进入方法）|`F8`|
|单步进入方法|`F7`|
|切换断点|`Ctrl + F8`|
|打开/隐藏项目面板|`Alt + 1`|
|打开终端|`Alt + F12`|
|关闭当前标签页|`Ctrl + F4`|
|打开设置|`Ctrl + Alt + S`|
