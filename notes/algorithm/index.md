# 算法与数据结构学习指南

## 概述

本模块系统性地介绍计算机科学中的核心算法和数据结构，涵盖从基础到高级的完整知识体系。每个章节都包含详细的理论讲解、代码实现和实战应用，帮助您建立坚实的算法基础，提升问题解决能力。

## 模块导航

### 1. [数据结构与算法基础](./数据结构与算法.md)
**学习目标**：建立数据结构与算法的整体认知框架

**核心内容**：
- 数据结构全体系知识图谱
- 基础数据结构（数组、链表、栈、队列）
- 高级数据结构（树、图、哈希表）
- 算法复杂度分析
- 数据结构选择原则


### 2. [算法应用与工程实践](./算法应用.md)
**学习目标**：掌握算法在实际工程中的应用技巧

**核心内容**：
- ACM/竞赛模式数据输入输出
- 算法优化技巧
- 工程实践中的算法选择
- 性能分析与调优
- 多语言实现对比

**适合人群**：有一定基础，关注算法实践应用的学习者

### 3. [搜索算法详解](./搜索.md)
**学习目标**：深入理解搜索算法的原理与应用

**核心内容**：
- 深度优先搜索（DFS）与广度优先搜索（BFS）
- 回溯算法与剪枝优化
- 双向搜索与启发式搜索
- 状态空间表示与压缩
- 搜索算法在游戏AI中的应用


### 4. [动态规划算法](./动态规划.md)
**学习目标**：掌握动态规划的核心思想和解题技巧

**核心内容**：
- 动态规划基本原理（最优子结构、重叠子问题）
- 状态定义与状态转移方程
- 背包问题系列（0-1背包、完全背包、多重背包）
- 区间DP、树形DP、状态压缩DP
- 动态规划优化技巧（斜率优化、四边形不等式）


### 5. [字符串算法](./字符串.md)
**学习目标**：掌握字符串处理的核心算法和技术

**核心内容**：
- 字符串匹配算法（KMP、BM、Sunday）
- 后缀数组与后缀自动机
- 回文串处理（Manacher算法）
- 字符串哈希与滚动哈希
- 字典树与AC自动机
